# 💰 Expense Manager

Это мой первый небольшой проект на Python — менеджер расходов в консоли.

Я сделал его, чтобы потренироваться в работе со списками, словарями и функциями.

---

## 📌 Что умеет программа

С помощью этой программы можно:

- Добавлять расходы (категория + сумма)
- Смотреть все расходы
- Считать общую сумму
- Находить самый большой и самый маленький расход
- Искать расходы по категории
- Удалять расходы
- Изменять уже добавленные расходы
- Смотреть простую статистику

---
